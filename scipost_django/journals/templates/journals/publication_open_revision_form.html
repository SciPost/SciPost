Open Publication to Revision(s)

{% include 'journals/_publication_summary.html' with publication=form.instance %}

Opening a Publication to revision(s) will do the following:

  • Reset the accepted Submission's status to "Awaiting resubmission".
  • Set the Publication's status to "Under revision".
  • Display the revision description as a warning in the Publication page.
  • Deprecate the latest Editorial Decision (keeping it as is otherwise).
{% include "bi/info-circle-fill.html" %} Note

If the current editor is unavailable, please first assign the latest submission to another editor before opening the Publication to revision(s).

This ensures that the new editor has access to all the necessary information to handle the revision process and that they are notified that the present Publication is now under revision.
